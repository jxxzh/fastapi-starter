# AI Herbal Formula System

This is a FastAPI project template generated by an AI assistant.

## Project Structure

- `server/`: Main application folder
  - `core/`: Core components like configuration (`config.py`).
  - `routers/`: API route modules (`items.py`).
  - `schemas/`: Pydantic schemas (`item.py`).
  - `main.py`: FastAPI application entry point.
- `.gitignore`: Specifies intentionally untracked files to ignore.
- `pyproject.toml`: Project metadata and dependencies for `uv`.
- `uv.lock`: Locked dependency versions.
- `README.md`: This file.

## Setup and Running the Application

This project uses [uv](https://github.com/astral-sh/uv) for dependency management.

### 1. Install Dependencies

If you haven't already, install the dependencies locked in `uv.lock`:
```bash
uv sync
```

### 2. (Optional) Create Environment File

The application can read settings from a `.env` file. You can create one in the root directory:

```env
# .env
ADMIN_EMAIL="admin@example.com"
```

### 3. Run the Application

You can run the development server using the following command:

```bash
uv run dev
```
The server will restart automatically when you make code changes.

### 4. Access the API

Once the server is running, you can access the interactive API documentation at:

- **Swagger UI:** [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)
- **ReDoc:** [http://127.0.0.1:8000/redoc](http://127.0.0.1:8000/redoc)
